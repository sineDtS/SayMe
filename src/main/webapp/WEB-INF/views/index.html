Welcome to SayMe Messenger

Welcome dear user :)

SayMe Messenger is a FREE browser messaging app used for searching friends, instant messaging, sharing photos. The app can be used to communicate with your friends. What the people like on the Web Messenger is the ability to find conversation very quick, the speed of the chat, the nice flat design and the big input areas.

You have to try this experience!!

It includes the following functions:

  • Sign-In
  • Admin / User roles
  • Redirect to custom error pages
  • View person & friend lists
  • View person's contact information
  • My Profile
